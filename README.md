# intercambio-estampas-mundia-2026
Pequeño sistema para intercambiar estampas 

# El Salvador

## Descripcion
Este sistema se creo con Gemini, adicional unas pequeñas modificacion de mi parte.
Se le puede sacar mayor provecho, pero solo es una prueba del funcionamiento de Github con Vercel

## Modo de uso
Entras al sitio, escribes tu numero de telefono de El Salvador, por ejemplo 71234567, 
seleccionas la estampa que ya tienes, doble clic si la tienes repetida y un clic mas si quiere que este libre nuevamente.
Cuando hayas terminado de seleccionar todas las estampas, entoncs haces clic en el boton de Compartir y ese enlace es el que compartiras con tus amigos.

Recuerda poner bien tu numero de telefono, porque sera el medio por el cual se comuniquen contigo.
